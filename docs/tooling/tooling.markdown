## Tooling to create UI5-Applications

UI5-Applications can - in theory - be created using only a code-editor. Whilst this is possible, most of the developers use a tooling for development which brings some mostly needed features out-of-the box. Using tools which are created for UI5 development is convenient and speeds up the development.

UI5 Applications can be created using a wide range of different tools and services. Some tools and services can be used free of charge and some need a subscription to be used.

### Standalone Tools

These tools are provided by SAP and need a subscription. These are standalone IDE's to be used for developing UI5-Applications.

- [SAP Business Application Studio](https://help.sap.com/viewer/product/SAP%20Business%20Application%20Studio/Cloud/en-US).

- [SAP WebIDE](https://developers.sap.com/topics/web-ide.html)

### Code-Editor Tooling

The tooling in this chapter are normally used in combination with a code-editor, i.e. VSCode. Using the package-manager, these tools are installed as dependency to develop UI5-Applications (dev-dependency).

Common tooling for UI5-Applications are:

- [UI5 Tooling](https://sap.github.io/ui5-tooling/)
- custom setup using grunt. grunt-openui5, ...
